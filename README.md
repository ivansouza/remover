# Remover Watermark Gemini

Remove marcas d'água do Gemini de imagens — **100% local**, nada sai do seu navegador.

🧠 Engine oficial do [Gemini Watermark Remover](https://github.com/GargantuaX/gemini-watermark-remover).

## Como usar

1. Acesse: **https://ivansouza.github.io/remover/**
2. Arraste uma imagem ou clique para selecionar
3. Pronto — a imagem sem watermark aparece do lado

## PWA

Instalável como aplicativo no celular/desktop. Funciona offline após o primeiro acesso.
